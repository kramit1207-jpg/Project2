# InsightProfile - AI-Powered Personality Analysis Application

InsightProfile is a full-stack personality analysis application that integrates Humantic AI API for behavioral data extraction and Google Gemini (via LangChain) for AI-powered personality insights. The application provides a visual dashboard displaying personality traits, strengths, and areas for growth.

## Features

- **LinkedIn Profile Analysis**: Input a LinkedIn profile URL to analyze personality
- **Humantic AI Integration**: Retrieves behavioral data using Humantic AI API
- **AI-Powered Insights**: Uses Google Gemini to analyze and generate personality summaries
- **Visual Dashboard**: Interactive radar chart showing Big Five personality traits
- **Strengths & Weaknesses**: Highlights 3 key strengths and 3 areas for growth
- **Professional UI**: Modern, responsive design with Tailwind CSS

## Technology Stack

### Backend
- **FastAPI**: Modern Python web framework
- **LangChain**: AI orchestration framework
- **Google Gemini**: AI model for personality analysis
- **Humantic AI API**: Behavioral data extraction
- **Uvicorn**: ASGI server

### Frontend
- **React**: UI library
- **Vite**: Build tool and dev server
- **Tailwind CSS**: Utility-first CSS framework
- **Recharts**: Charting library for radar visualization
- **Lucide React**: Icon library
- **Axios**: HTTP client

## Prerequisites

## Setup Instructions

### Backend Setup

1. **Navigate to the backend directory:**
   ```bash
   cd backend
   ```

2. **Create a virtual environment (if not using the setup script):**
   ```bash
   python -m venv venv
   ```

3. **Activate the virtual environment:**
   - Windows:
     ```bash
     venv\Scripts\activate
     ```
   - Linux/Mac:
     ```bash
     source venv/bin/activate
     ```

4. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

   **OR use the setup script (Windows):**
   ```bash
   setup_env.bat
   ```

5. **Create a `.env` file:**
   ```bash
   copy .env.example .env
   ```

6. **Edit `.env` file and add your API keys:**
   ```
   HUMANTIC_API_KEY=your_humantic_api_key_here
   GOOGLE_API_KEY=your_google_api_key_here
   ```

7. **Start the backend server:**
   ```bash
   uvicorn main:app --reload
   ```

   The backend will run on `http://localhost:8000`

### Frontend Setup

1. **Navigate to the frontend directory:**
   ```bash
   cd frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

   The frontend will run on `http://localhost:3000`

## Usage

1. **Start both backend and frontend servers** (see Setup Instructions above)

2. **Open your browser** and navigate to `http://localhost:3000`

3. **Enter a LinkedIn profile URL** in the search bar (e.g., `linkedin.com/in/username`)

4. **Click "Analyze Profile"** and wait for the analysis to complete (may take 30-60 seconds)

5. **View the results:**
   - **Personality Summary**: 2-sentence summary generated by Gemini
   - **Personality Profile**: Radar chart showing Big Five personality traits
   - **Strengths**: 3 key strengths identified
   - **Areas for Growth**: 3 areas for improvement

## API Endpoints

### POST `/api/analyze`

Analyzes a LinkedIn profile and returns personality insights.

**Request Body:**
```json
{
  "linkedin_url": "linkedin.com/in/username"
}
```

**Response:**
```json
{
  "analysis": {
    "summary": "Personality summary text...",
    "strengths": ["Strength 1", "Strength 2", "Strength 3"],
    "weaknesses": ["Weakness 1", "Weakness 2", "Weakness 3"]
  },
  "raw_scores": {
    "openness": 75.5,
    "conscientiousness": 68.2,
    "extraversion": 82.1,
    "agreeableness": 71.3,
    "neuroticism": 45.8
  }
}
```

### GET `/health`

Health check endpoint.

**Response:**
```json
{
  "status": "healthy"
}
```

## Troubleshooting

### Backend Issues

**Issue: Module not found errors**
- Solution: Ensure virtual environment is activated and all dependencies are installed

**Issue: API key errors**
- Solution: Verify `.env` file exists and contains valid API keys

**Issue: CORS errors**
- Solution: Ensure backend is running on port 8000 and frontend on port 3000

**Issue: Humantic API errors**
- Solution: Check your API key is valid and you haven't exceeded rate limits
- Note: There's a 30-45 second wait between profile creation and fetching to optimize quota

### Frontend Issues

**Issue: Cannot connect to backend**
- Solution: Verify backend is running on `http://localhost:8000`
- Check browser console for CORS or connection errors

**Issue: Dependencies not installing**
- Solution: Try deleting `node_modules` and `package-lock.json`, then run `npm install` again

**Issue: Styles not loading**
- Solution: Ensure Tailwind CSS is properly configured and PostCSS is set up

### General Issues

**Issue: Analysis takes too long**
- This is normal - the process includes:
  1. Creating profile in Humantic AI (~5-10 seconds)
  2. Waiting 35 seconds for processing
  3. Fetching profile (~5 seconds)
  4. Analyzing with Gemini (~5-10 seconds)
- Total time: ~50-60 seconds

**Issue: Invalid LinkedIn URL error**
- Solution: Use the format `linkedin.com/in/username` or full URL `https://www.linkedin.com/in/username`

## Rate Limits

### Humantic API
- **Profile Creation (LinkedIn URLs):**
  - Warning: 20 calls in 2 minutes or 50 in 10 minutes
  - Block: 30 calls in 2 minutes or 100 in 10 minutes (30-minute block)

- **Profile Fetching:**
  - Warning: 50 calls in 2 minutes or 100 in 10 minutes
  - Block: 100 calls in 2 minutes or 200 in 10 minutes (30-minute block)

Please respect these limits to avoid temporary API key blocks.
